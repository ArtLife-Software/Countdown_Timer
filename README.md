# Countdown Timer

Windows 桌面倒數計時器，支援多視窗同時運作、彈性提醒與語音倒數。常駐系統匣，開箱即用。

---

## 系統需求

- Windows 10 x64 以上
- .NET Framework 4.x

---

## 安裝與執行

不需安裝，解壓縮後直接執行 `Countdown Timer.exe`。

首次啟動會自動建立預設設定檔，並開啟一個計時器視窗。

---

## 目錄結構

```
Countdown Timer.exe
Config\                  # 計時器設定檔（.ini）
Location\                # 視窗位置記錄
Sound_effects\           # 自訂音效與語音倒數檔案
Countdown_Timer.ico      # 自訂圖示（可選）
```

---

## 功能說明

### 計時器視窗

每個計時器為獨立視窗，可自由縮放與拖移。視窗顯示：

- 倒數時間（時：分：秒，可開啟小數）
- 進度條與百分比
- 已經過時間（正數計時）
- 預計結束時間
- 操作按鈕（開始 / 暫停 / 停止 / 重啟 / 繼續 / 初始化）

支援**負計時模式**，歸零後繼續往負數倒數。

### 多視窗管理

- 可同時開啟多個計時器，每個獨立運作
- 相同設定檔可開啟多個實例，標題列自動加上 `#N` 編號
- **自動排列**：一鍵將所有視窗整齊排列於螢幕工作區

### 系統匣

程式常駐系統匣，右鍵選單提供：

- 新增計時器
- 啟動倒數計時器（LaunchForm）
- 預設設定
- 自動排列計時器
- 關閉所有計時器
- 結束程式

### 設定

每個計時器有獨立設定，分為以下頁籤：

**基本**
- 時長（時 / 分 / 秒）
- 負計時、小數顯示（1–3 位）、正數計時、結束時間顯示

**外觀**
- 字型、前景色、背景色、進度條顏色
- 視窗縮放比例（25%–200%）
- 最上層顯示

**間隔提醒（AI）**
每隔固定時間觸發一次，支援秒 / 分 / 時單位。

**剩餘提醒（AR）**
剩餘時間低於門檻時觸發，持續提醒至歸零。

**結束提醒（AE）**
計時結束時觸發，持續提醒。

各提醒皆可設定：
- 音效：預設音效 / 自訂 WAV / 無聲
- 顏色切換：前景色 / 背景色 / 進度條色可各自啟用

**語音倒數（VO）**
最後 3 / 5 / 10 秒播放語音倒數檔（`Sound_effects\Countdown_voice_XX.wav`）。

### 啟動視窗（LaunchForm）

集中管理所有計時器設定檔：

| 功能 | 說明 |
|------|------|
| 啟動 | 開啟選取的計時器視窗 |
| 啟動並開始計時 | 開啟並立即開始 |
| 啟動數量 | 同一設定檔開啟多個實例 |
| 再製 | 複製設定檔並開啟 |
| 設定 | 開啟單一設定視窗 |
| 批次設定 | 同時修改多個計時器的設定 |
| 更名 | 重新命名設定檔 |
| 關閉 / 刪除 | 關閉視窗或刪除設定檔 |

**批次控制計時器**（對選取或全部已開啟的計時器）：開始、重啟、暫停、繼續、停止、初始化。

## 音效檔案

將 WAV 檔案放入 `Sound_effects\` 目錄：

| 檔名 | 用途 |
|------|------|
| `Interval.wav` | 間隔提醒預設音效 |
| `Remainder.wav` | 剩餘提醒預設音效 |
| `End.wav` | 結束提醒預設音效 |
| `Countdown_voice_03.wav` | 語音倒數 3 秒版 |
| `Countdown_voice_05.wav` | 語音倒數 5 秒版 |
| `Countdown_voice_10.wav` | 語音倒數 10 秒版 |

未放置檔案時，自動退回使用 Windows 內建通知音效。

---

## 自訂圖示

將 `Countdown_Timer.ico` 放於程式同目錄下即可套用至視窗與系統匣。
